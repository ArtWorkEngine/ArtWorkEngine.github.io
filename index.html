The Quillfull Language

Quillfull is a custom scripting language designed specifically for ArtWork Engine. With clean, readable syntax and powerful features, it makes game development accessible to both beginners and experienced developers.

Focus on your game logic while Quillfull handles the complex underlying systems. From AI behavior to physics simulations, Quillfull has you covered.

Learn Quillfull
entity Player {
  sprite = "player.png";
  health = 100;
  speed = 5;
  function onUpdate() {
    if (keyPressed("space")) {
      jump();
    }
  }
}
scene GameWorld {
  background = "forest.png";
  player = Player();
}
// Simple, readable code for complex game logic
